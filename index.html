Help

Configuration

  1. Open rocketleague/TAGame/Config/DefaultStatsAPI.ini
  2. Change the packet send rate from 0 to above 0 (Max of 120 recommended)
  3. Make sure your port is 49123
  4. Save and restart RL

How do I add the overlay to my streaming software?

  1. Open your Streaming Software of choice
  2. Create a new browser source
  3. Set the source to: http://localhost:3000
  4. Set the resolution to: 1920px (width) by 1080px (height) Standard 1080p
  5. Click OK
  6. Right click the source you just made
  7. Go to the "Transform" option
  8. Then click "Fit to screen"
